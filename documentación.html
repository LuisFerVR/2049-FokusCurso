Donde: elemento: Es el elemento HTML al cual queremos asociar el evento. evento: Es un string que representa el tipo de evento que queremos capturar. callback: Es la función que se llamará cuando ocurra el evento.

Tipos de Eventos
Eventos de clic (click)

Los eventos de clic son algunos de los más utilizados en el desarrollo web. Ocurren cuando el usuario hace clic en un elemento específico de la página, como un botón, un enlace o incluso una imagen. Podemos usar el evento click para ejecutar acciones cuando el usuario interactúa con estos elementos.

Eventos de envío de formulario (submit)

Cuando tenemos un formulario en nuestra página, podemos usar el evento submit para capturar el envío del formulario por parte del usuario. Esto nos permite realizar acciones como validar los datos ingresados antes de enviarlos al servidor.

Eventos de teclado (keydown, keyup, keypress)

Los eventos de teclado nos permiten responder a las acciones del usuario en el teclado, como presionar o soltar una tecla específica. Hay tres tipos principales.

    Eventos de teclado:
  • keydown: Ocurre cuando se presiona una tecla.
  • keyup: Ocurre cuando se suelta una tecla.
  • keypress: Ocurre cuando se presiona una tecla y aún no se ha soltado.
Eventos de enfoque (focus, blur)

Los eventos de enfoque se utilizan cuando queremos capturar cuándo un elemento recibeo pierde el enfoque. El evento focus ocurre cuando el elemento gana el enfoque (por ejemplo, cuando hacemos clic en un campo de formulario), mientras que el evento blur ocurre cuando el elemento pierde el enfoque.

Manipular elementos
getAttribute

se usa para obtener el valor de un atributo específico de un elemento HTML. Toma un argumento, que es el nombre del atributo del cual queremos recuperar el valor.

setAttribute

se utiliza para definir o modificar el valor de un atributo en un elemento HTML. Acepta dos argumentos: el primero es el nombre del atributo que queremos definir o modificar, y el segundo es el valor que queremos asignar a ese atributo. Si el atributo ya existe, el método setAttributelo sobrescribirá; de lo contrario, creará un nuevo atributo.

hasAttribute

El método hasAttribute se utiliza para verificar si un elemento tiene un atributo específico. Toma un argumento, que es el nombre del atributo que queremos verificar. El método devolverá true si el atributo existe y false si el atributo no está presente.

removeAttribute

se usa para eliminar un atributo específico de un elemento HTML. Toma un argumento, que es el nombre del atributo que queremos eliminar.

classList

es una propiedad de JavaScript que representa una lista de clases CSS. Proporciona métodos que facilitan la adición, eliminación y verificación de clases, haciendo que la manipulación de clases CSS sea más eficiente y menos propensa a errores de programación.

Audio

El objeto 'Audio' es parte de HTML5 y proporciona una interfaz fácil y poderosa para reproducir y controlar archivos de audio en tiempo real.

Controles básicos de audio
play()

inicia la reproducción del audio.

pause

pausa la reproducción del audio.

currentTime

propiedad que devuelve o establece la posición actual de reproducción del audio, en segundos.

volume

propiedad que devuelve o establece el nivel de volumen del audio, variando de 0 a 1.

loop

propiedad que repite la reproducción del audio pniendo como true el valor de esta propiedad.

Construcción de un temporizador
Porpiedad Nodo
parentNode

se utiliza para acceder al nodo padre de un elemento en el DOM. A través de ella, podemos navegar por el árbol del DOM hacia el nodo padre del elemento actual.

childNodes

se utiliza para acceder a todos los nodos hijos de un elemento en el DOM. Retorna una lista de nodos, incluyendo nodos de texto y elementos HTML.

nextElementSibling

permite acceder al siguiente hermano (nodo adyacente) de un elemento en el DOM.

previousElementSibling

similar a nextElementSibling , pero permite acceder al hermano anterior (nodo adyacente) de un elemento en el DOM.
